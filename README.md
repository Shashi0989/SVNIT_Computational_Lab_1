# Python problems solved in Computational Physics Lab-NIT Surat

## 📌 Overview

This repository contains a comprehensive set of **computational physics programs implemented in Python**, covering numerical methods, classical physics simulations, stochastic processes, and data visualization.

The lab integrates **mathematical modeling, numerical algorithms, and physical systems** into interactive and visual simulations.

---

## 🚀 Modules Included

### 🔹 Random Number Generation & Statistics

* Mid-square method, Fibonacci generators, built-in RNG comparison 
* Distribution visualization and histogram animation 
* Cauchy distribution sampling using rejection method 

---

### 🔹 Monte Carlo Methods

* Estimation of π with live animation 
* Nuclear decay simulation (analytic + Monte Carlo comparison) 

---

### 🔹 Numerical Methods

* Newton–Raphson root finding with convergence analysis 
* Numerical integration:

  * Trapezoidal rule
  * Simpson’s 1/3 rule
  * Simpson’s 3/8 rule 
* Runge–Kutta methods (2nd, 3rd, 4th order) for ODEs 

---

### 🔹 Classical Physics Simulations

* Projectile motion (trajectory + animation) 
* Electric dipole field and potential visualization 
* Sinusoidal waveform plotting and animation 

---

### 🔹 Stochastic Processes

* 2D random walk simulation with statistical analysis 

---

### 🔹 Circuit Analysis

* Kirchhoff’s laws solved using:

  * SciPy linear solver
  * LU decomposition
  * Cramer’s rule 

---

## 🧠 Concepts Covered

* Monte Carlo methods
* Random number generation
* Numerical integration
* Root-finding algorithms
* Ordinary differential equations
* Classical mechanics
* Electrodynamics
* Statistical simulations

## 🛠️ Requirements

Install required libraries:

```bash
pip install numpy scipy matplotlib sympy
```

---

## ▶️ How to Run

Run any script individually:

```bash
python filename.py
```

Example:

```bash
python Newton_Raphson.py
python Random_Walk.py
python Nuclear_Decay.py
```
